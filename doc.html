Getting Started

Shop Management is a native x64 Windows application designed for jewellery businesses to maintain customer records offline with zero cloud dependency.

Installation

  1. Download the latest .exe setup from the home page.
  2. Run the installer. If Windows SmartScreen appears, click "More Info" and "Run Anyway" (as the app is currently self-signed).
  3. The app will launch automatically once the installation is complete.

WhatsApp Core Integration

The app uses a built-in messaging engine to automate customer outreach.

How to use:

  • Navigate to the Campaigns tab.
  • Select your target customer group.
  • Use variables like {name} or {balance} in your template.

Data & Privacy

Your data is stored locally using the NeDB engine. You can find your database file at: 

C:\Users\YourName\AppData\Roaming\shop-management\customers.db

Pro Tip: Always back up this file before performing a fresh system install.
